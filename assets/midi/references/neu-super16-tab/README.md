# Neu! Super / Super 16 Tab Translation MIDI

This folder contains a practical MIDI translation of the user-supplied tab text for Neu!'s "Super" ("Super 16").

Files:

- `neu-super16-tab-translation.mid`

Track layout:

- Rhythm Guitar (clean): fast eighth-note chord/power-chord chug with tab-inspired fill gestures
- Fuzz Riff: lower root-driven doubling for drive
- Section Markers: Verse/Bridge markers as MIDI text events

Tempo:

- 160 BPM (fast, guitar-driven feel)

Script:

- `tools/generation/generate_neu_super16_tab_midi.py`

Notes:

- This is a tab-driven arrangement translation for starter vocabulary, not a full studio-transcription.
