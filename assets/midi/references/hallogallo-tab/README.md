# Hallogallo Tab Translation MIDI

This folder contains a generated MIDI interpretation of the user-supplied Hallogallo tab excerpt.

Files:

- `hallogallo-tab-translation.mid`

Track layout:

- Track 1: Bass riff loop (A-string fret 7 pattern + periodic fret-5 variation)
- Track 2: Guitar 1 power-chord doubling of bass rhythm
- Track 3: Guitar 2 motif entries based on tab phrase

Generation script:

- `tools/generation/generate_hallogallo_tab_midi.py`

Notes:

- This is a practical MIDI translation of the provided tab text, not a full transcription of the original recording.
- Tempo set to 124 BPM for preview/reference.
